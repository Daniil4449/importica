## 🚀 Як запустити на GitHub Pages

1. Створити репозиторій на GitHub: `importica`
2. Завантажити всі файли
3. Settings → Pages → Source: `main` branch → `/ (root)`
4. Підключити домен: `importica.com.ua`

### Підключення домену importica.com.ua:
1. У реєстратора домену додати DNS записи:
   ```
   A     @     185.199.108.153
   A     @     185.199.109.153
   A     @     185.199.110.153
   A     @     185.199.111.153
   CNAME www   YOUR-USERNAME.github.io
   ```
2. В GitHub Pages → Custom domain → ввести `importica.com.ua`
3. Увімкнути `Enforce HTTPS`
